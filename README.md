## 功能点：
- 所有变量存储在localstroage里
- 初始化，加载localstroage数据，没有则为空
- 可以添加，从页面里环境变量中选取或者手动添加
- 可以删除，任意一条或者多条localstorage中的值
- 可以修改localstorage中的值
- 填充页面，单选或者多选，有该key则修改，没有则添加
- 清空所有localstorage中值


## 技术知识点：
- ES6
- webpack
- babel
- less
- vue組件通信
- chrome插件--扩展和content_script通信（长链接和短链接）
- chrome插件--content_script对tab内页面dom的操作
- chrome插件--本地存储使用方法
